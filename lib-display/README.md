# Library Display
## Cross-platform Open Source C++

Supported displays:

- OLED display 128x32 SSD1306 I2C
- OLED display 128x64 SSD1306 I2C
- LCD Module TC1602A / PCF8574T I2C
- LCD Module 2004 / PCF8574T I2C
